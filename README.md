## 🚀 推荐使用方式

1. **快速了解范围** → `计算机组成原理复习计划.md`，看各章重要度和题型预测
2. **系统学知识点** → `知识点精讲.md`，逐章过核心概念和公式
3. **刷题查漏补缺** → `真题与解析.md`，先自己做再看解析
4. **考前速查** → 用 `计算机组成原理复习计划.md` 最后的自检清单逐条排查

## 🔥 重点速览

```
考试最核心的 6 个板块：
1. 补码运算 + IEEE 754 浮点数   — 计算题必考
2. Cache 地址映射与替换算法      — 分析题必考
3. 单总线 CPU 微操作序列         — 综合题必考
4. 流水线时空图与性能计算        — 计算/分析题必考
5. 中断 vs DMA 对比              — 简答题必考
6. CRC/海明校验码                — 计算题常考
```
