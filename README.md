## 📖 内容概览

本仓库提供了一份**结构化的计算机组成原理期末复习计划**，涵盖 9 大章节：

| 章节 | 核心内容 |
|:---:|------|
| 1 | 计算机系统概论（层次结构、性能指标、CPU 性能公式） |
| 2 | 数据信息表示（机器码、IEEE 754 浮点数、海明/CRC 校验） |
| 3 | 运算方法与运算器（补码运算、浮点加减法、Booth 乘法） |
| 4 | 存储系统（Cache 映射与替换、虚拟存储器、主存扩展） |
| 5 | 指令系统（寻址方式、CISC vs RISC、MIPS 指令格式） |
| 6 | 中央处理器（数据通路、硬布线/微程序控制器、中断处理） |
| 7 | 指令流水线（时空图、流水线冲突、性能分析） |
| 8 | 总线系统（总线仲裁、总线定时、总线结构） |
| 9 | 输入输出系统（中断、DMA、通道方式对比） |

## ✨ 复习计划特点

- 📅 **7 天速通安排** — 每天复习哪些内容一目了然，支持 checkbox 打卡
- 🔥 **6 大重点专题** — 数据校验、IEEE 754、Cache 映射、单总线通路、流水线分析、中断 vs DMA
- 🧠 **记忆口诀** — 浮点加减法、Cache 映射、I/O 方式等速记技巧
- 📝 **8 种必做题型** — 覆盖高频计算/分析题
- ✅ **14 条考前自检** — 确保无知识盲区
- 📊 **题型预测表** — 各章节可能在哪种题型出现

## 🚀 使用方式

1. 打开 `计算机组成原理复习计划.md`
2. 按天完成复习任务，勾选 checkbox
3. 重点关注 **🔥 重点难点专题** 部分
4. 考前用 **✅ 自检清单** 逐条排查

## ⚠️ 注意事项

- 文档中标注 `*` 的章节为选学/扩展内容，考试概率较低
- 第 3.4.1 节「恢复余数除法」明确不考
- 历年真题是最高效的复习方式，建议至少做近 3 年真题
