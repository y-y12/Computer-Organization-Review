## 📅 每日复习计划（7天速通版）

### 第 1 天：计算机系统概论 + 数据信息表示（上）

**第 1 章 · 计算机系统概论**
- [ ] 计算机发展历程中的重要节点（摩尔定律、集成电路工艺）
- [ ] 计算机硬件系统组成（五大部件：运算器、控制器、存储器、输入设备、输出设备）
- [ ] 计算机软件系统分类（系统软件 vs 应用软件）
- [ ] **计算机系统的层次结构**（微程序级 → 机器语言级 → 操作系统级 → 汇编语言级 → 高级语言级）
- [ ] 各层之间的关系、软硬件逻辑功能等价性
- [ ] **性能指标**：主频、CPI、MIPS、MFLOPS、CPU 性能公式 $T = N \times CPI \times T_c$
- [ ] 掌握 CPU 性能公式的应用计算

**第 2 章 · 数据信息的表示（前半）**
- [ ] 数的机器码表示：**原码、反码、补码、移码**的定义、转换、表示范围

## 🔥 重点难点专题

### 专题一：数据校验计算
- 海明码：给出数据位，能编出海明码；给出带错海明码，能定位错误位
- CRC 校验：给定生成多项式，能计算 CRC 校验码；能判断接收数据是否正确

### 专题二：浮点数 IEEE 754
- 单精度（32 位）与双精度（64 位）格式
- 十进制 ↔ IEEE 754 浮点数的相互转换
- 浮点加减法的完整流程（对阶、尾数运算、规格化、舍入、溢出判断）

### 专题三：Cache 地址映射与性能计算
- 直接映射 / 全相联 / 组相联：给定主存地址格式，划分 Tag / Index / Offset
- 命中率、平均访问时间计算
- 替换算法模拟（LRU 栈模拟）

### 专题四：单总线 CPU 数据通路分析
- 给定指令，能写出在单总线结构上的微操作序列（取指、间址、执行、中断）
- 能分析硬布线控制器和微程序控制器的设计方法

### 专题五：流水线时空图与性能分析
- 画流水线时空图（标注冲突、气泡插入）
- 吞吐率 $TP = \frac{n}{T_k}$、加速比 $S_p = \frac{T_1}{T_k}$、效率 $E = \frac{S_p}{k}$
- 区分三种冲突类型并给出解决方案

### 专题六：中断与 DMA 对比
| 特性 | 程序中断 | DMA |
|------|---------|-----|
| 数据传送单位 | 字/字节 | 数据块 |
| 传送控制方式 | 程序控制（CPU 执行中断服务程序） | 硬件控制（DMA 控制器） |
| CPU 参与程度 | CPU 全程参与 | 仅开始和结束时参与 |
| 传送速度 | 慢 | 快 |
| 适用场景 | 低速 I/O 设备 | 高速 I/O 设备 |
| 对 CPU 的影响 | 打断 CPU 现行程序 | 窃取总线周期 |

## 🧠 记忆口诀辅助

**计算机层次结构（由上到下）：**
> 应用 → 高级语言 → 汇编语言 → 操作系统 → 机器语言 → 微程序 → 硬件逻辑

**浮点加减法五步：**
> 对阶 → 尾数加 → 规格化 → 舍入 → 判溢出
>
> 记：**"对尾格舍判"**（对尾规舍判）

**Cache 地址映射三种方式：**
> **直全组** — 直接映射、全相联、组相联

**总线仲裁方式（集中式）：**
> **菊轮独** — 菊花链、轮询、独立请求

**I/O 控制方式（按速度/复杂度递增）：**
> **查询 → 中断 → DMA → 通道**
>
> 记：**"查中 D 通"**
